The CERIcar

Présentation du projet :

Ce projet a été réalisé lors du premier semestre de L3. Il avait pour but le développement d'une application web permettant aux utilisateurs de proposer et réserver des trajets de covoiturage. Les informations sur les trajets ainsi que sur les utilisateurs étaient stockées dans une base de données SQL.

L'application est codée en html, php, css, java script et est organisée selon une architechture MVC. Elle prend également en charge des requêtes Ajax.

Fonctionnalités :

  • Création de compte et identification.
  • Recherche de trajets selon des critères donnés, tout en proposant des correspondances.
  • Ajout de nouveaux trajets.
  • Consultation du profil avec
